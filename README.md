# Bridging Bioengineering and Control Systems for Fermentation Research

Welcome! This repository is dedicated to advanced bioprocess engineering, process control, and machine learning applications in biotechnology. As a cross-disciplinary researcher in **Bioengineering** and **Electrical Engineering (EE)**, I focus on system identification, soft-sensing, and optimization of complex fermentation processes.

---

## 📊 Datasets & Benchmarks

We gather, structure, and maintain high-quality, validated dataset resources for bioengineering and fermentation research. These resources are specifically tailored for EE/Data Science researchers looking for reliable time-series data.

👉 **Please visit our [Dataset Directory](./dataset/)** for detailed profiles, variable breakdowns (CPPs vs. CQAs), and official download links (including the industrial-scale **IndPenSim** benchmark).

---

## 🛠️ Research Focus & Framework

This repository will be continuously updated with code and documentation covering:
* **Soft-Sensors:** Real-time prediction of offline biomass and product concentration.
* **Process Monitoring:** Fault Detection and Isolation (FDI) in multi-batch bio-systems.
* **Optimal Control:** Reinforcement learning and Model Predictive Control (MPC) for feeding strategies.
